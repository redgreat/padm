    前端介绍

    这是一个基于Vue3的Vite项目，使用Vue3、Vite、TypeScript、Pinia、Vue-router、Element Plus、Axios等框架。

    npm init vue@latest 创建项目 cd frontend 进入项目 npm install 安装依赖 npm run dev 启动项目

    推荐IDE设置

    VSCode + Volar (and disable Vetur).

    项目配置

    查看 Vite Configuration Reference.

    项目启动

    初始化依赖

    npm install

    启动开发环境

    npm run dev

    构建项目

    npm run build

    项目检查

    [ESLint](https://eslint.org/)

npm run lint
